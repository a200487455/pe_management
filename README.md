## 项目介绍
基于springboot+vue的宠物领养管理系统：前端 vue2、elementui，后端 maven、springmvc、spring、mybatis；角色分为管理员、用户；集成宠物信息、宠物领养、宠物用品、购物车、订单、等功能于一体的系统。
